## 🚀 快速开始

### 1. 安装 OpenClaw

```bash
# macOS / Linux / WSL2
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### 2. 运行配置向导

```bash
openclaw onboard --install-daemon
```

### 3. 打开控制面板

```bash
openclaw dashboard
```

详细步骤请查看 [完整安装指南](openclaw-install-guide.md)
